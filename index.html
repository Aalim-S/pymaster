What is a List?

A list in Python is an ordered, mutable collection of items. Lists can hold items of any data type — even other lists!

💡

Lists are one of the most powerful and frequently used data structures in Python. Mastering them is essential for any Python developer.

# Creating a list
fruits = ["apple", "banana", "mango"]
print(fruits) # ['apple', 'banana', 'mango']

# Mixed types list
mixed = [1, "hello", 3.14, True]

Accessing Elements

Use indexing to access elements. Python uses zero-based indexing — the first element is at index 0.

fruits = ["apple", "banana", "mango"]

print(fruits[0]) # apple (first)
print(fruits[-1]) # mango (last)
print(fruits[1:3]) # ['banana', 'mango'] (slicing)

Common List Methods

.append(x)
Add item to end
.remove(x)
Remove first occurrence
.sort()
Sort the list in place
.len(x)
Get list length
